== Description ==

This pulgin can stop particular users whose data accords with the imformation setted and saved. | Wordpress垃圾评论过滤可以以昵称 IP 邮箱 网址 内容片断为根据指定评论人员黑名单，并阻止他们提交评论。同时，评论次数超过设定的范围就会暂时禁止评论。当然，还将全英文、全部复制文章内容作为内容的评论进行过滤。最后为了安全起见，还提供了禁止特定IP用户访问站点！！
